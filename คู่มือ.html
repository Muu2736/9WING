วิธีผูกคีย์กับ Username และ Password

ทำครั้งแรกเพื่อสร้างข้อมูลเข้าสู่ระบบและเชื่อมคีย์เข้ากับบัญชีของคุณ

ACCOUNT ⌄

  1. เปิดหน้าล็อกอิน

    เปิดโปรแกรม 9WING แล้วกดปุ่ม “ผูกคีย์กับไอดี” ที่อยู่ด้านล่างแบบฟอร์มเข้าสู่ระบบ

  2. กรอกคีย์ที่ได้รับ

    กรอก Key ให้ครบถ้วนตามที่ได้รับจากผู้ดูแล ระวังตัวพิมพ์เล็ก–ใหญ่และช่องว่างส่วนเกิน

  3. ตั้ง Username และ Password

    ตั้งชื่อผู้ใช้และรหัสผ่านของตัวเอง จากนั้นกรอกรหัสผ่านยืนยันอีกครั้งให้ตรงกัน

  4. กดยืนยันการผูกคีย์

    ตรวจสอบข้อมูลทั้งหมดแล้วกดปุ่มยืนยัน รอจนระบบแจ้งว่า ผูกคีย์สำเร็จ

  5. กลับมาเข้าสู่ระบบ

    ใช้ Username และ Password ที่สร้างไว้เข้าสู่ระบบได้ทันที โดยไม่ต้องนำ Key มากรอกซ้ำทุกครั้ง

การตั้งค่า LDPlayer

ตั้งค่าเครื่องจำลองให้ลื่นและเหมาะกับการใช้งาน GG Power และ 9WING

EMULATOR ⌄
⚙️

CPU

4 Cores

หากเครื่องสเปกต่ำให้เริ่มที่ 2 Cores และอย่าเลือกเกินจำนวนคอร์ที่เครื่องรองรับ

💾

RAM

4096 MB

เครื่องที่มี RAM 16 GB ขึ้นไปสามารถปรับเป็น 6144 MB เพื่อเพิ่มความลื่นได้

▣

ความละเอียด

1280 × 720

เลือกโหมดแท็บเล็ตหรือกำหนดเอง พร้อมตั้ง DPI ที่ 240 เพื่อให้ปุ่มและหน้าจอพอดี

🎞️

เฟรมเรต

60 FPS

เปิดโหมดเฟรมเรตสูงเฉพาะเมื่อคอมพิวเตอร์ทำงานได้เสถียรและไม่เกิดอาการกระตุก

🎮

โหมดกราฟิก

OpenGL

หากภาพดำ ภาพแตก หรือเปิดเกมไม่ได้ ให้สลับเป็น DirectX แล้วเปิด LDPlayer ใหม่

🧹

ลดภาระเครื่อง

ปิดเสียงและแอปพื้นหลัง

ปิดโปรแกรมที่ไม่จำเป็น และหลีกเลี่ยงการเปิด LDPlayer หลายหน้าจอเกินกำลังเครื่อง

🔧 ขั้นตอนการตั้งค่า

  1. เปิดเมนูตั้งค่า

    เปิด LDPlayer แล้วกดรูปเฟืองด้านขวาบนของหน้าต่าง

  2. เข้าเมนูขั้นสูง

    ตั้งค่าความละเอียด CPU และ RAM ตามค่าที่แนะนำด้านซ้าย

  3. ตั้งค่าการแสดงผล

    เลือก 60 FPS และเริ่มด้วย OpenGL หากมีปัญหาค่อยเปลี่ยนเป็น DirectX

  4. บันทึกและรีสตาร์ต

    กดบันทึก จากนั้นปิดและเปิด LDPlayer ใหม่หนึ่งครั้งก่อนเริ่มใช้งาน

  5. ทดสอบก่อนเปิดระบบ

    เข้าเกมให้เรียบร้อย ตรวจสอบว่าภาพไม่ค้างและการควบคุมตอบสนองปกติ

หมายเหตุ: ค่าที่แนะนำเป็นค่าเริ่มต้นสำหรับเครื่องทั่วไป หากเครื่องกระตุกหรือร้อนมาก ให้ลด CPU, RAM หรือ FPS ลงทีละระดับ และควรเปิด VT ใน BIOS เพื่อประสิทธิภาพที่ดีขึ้น

หากรุ่นที่ใช้งานมีเมนูความละเอียด ให้เลือก 720p และหมุนหน้าจอเป็นแนวนอนก่อนเข้าเกม

🌐

เครือข่าย

เลือกเซิร์ฟเวอร์ใกล้ไทย

เลือกโซนที่มีค่า Ping ต่ำและใช้อินเทอร์เน็ตที่เสถียร เพื่อลดอาการหน่วงหรือหลุด

🔐

สิทธิ์แอป

อนุญาตสิทธิ์ที่จำเป็น

เปิดสิทธิ์แสดงทับแอปอื่น การเข้าถึงไฟล์ การแจ้งเตือน และ Accessibility เมื่อโปรแกรมร้องขอ

🔋

การทำงานเบื้องหลัง

ไม่จำกัดแบตเตอรี่

หากมีเมนูประหยัดพลังงาน ให้ตั้งเกมและโปรแกรม 9WING เป็นไม่จำกัด เพื่อป้องกันระบบหยุดเอง

🧹

ลดภาระ Cloud Phone

ปิดแอปที่ไม่ใช้

เปิดเฉพาะเกม GG Power และ 9WING หลีกเลี่ยงการติดตั้งแอปจำนวนมากหรือเปิดหลายงานพร้อมกัน

☁️ ขั้นตอนเริ่มใช้งาน

  1. เข้าสู่ระบบ Redfinger

    เปิดแอปหรือเว็บไซต์ Redfinger แล้วเลือก Cloud Phone ที่ต้องการใช้งาน

  2. ตั้งภาษาและเวลา

    ตั้งภาษาไทยและเขตเวลา GMT+7 หรือ Bangkok เพื่อให้เวลาในระบบตรงกับประเทศไทย

  3. ติดตั้งไฟล์ที่จำเป็น

    ติดตั้งตัวเกม GG Power และแอป 9WING จากแหล่งดาวน์โหลดที่ผู้ดูแลกำหนด

  4. เปิดสิทธิ์ให้ครบ

    อนุญาตสิทธิ์ตามที่แอปร้องขอ โดยเฉพาะการแสดงทับแอปอื่นและ Accessibility

  5. เข้าเกมก่อนเปิดระบบ

    ล็อกอินเกม เลือกตัวละคร และรอให้เข้าแผนที่เรียบร้อย จากนั้นจึงเปิด GG Power หรือ 9WING

  6. ทดสอบการทำงาน

    ตรวจสอบว่าปุ่มลอยแสดงครบ เกมไม่ค้าง และระบบยังทำงานต่อเมื่อปิดหน้าจอควบคุม Redfinger

ข้อควรระวัง: ระหว่างระบบกำลังทำงาน ห้ามกดปิดเครื่อง รีสตาร์ต หรือล้างข้อมูล Cloud Phone และไม่ควรย้ายบัญชีไปมาหลายเครื่องในวันเดียวกัน เพราะอาจกระทบสิทธิ์การใช้งาน
